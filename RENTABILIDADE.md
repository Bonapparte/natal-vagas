# Natal Vagas Pro - Business Setup

## Estratégia de Ganhos Rápidos

### 1. Vagas Premium (R$ 49,90 / 15 dias)
- Implementar via Stripe (Checkout grátis para começar).
- As vagas ficam no topo da lista com selo de destaque.

### 2. Base de Currículos VIP (R$ 19,90 / mês)
- Empresas pagam para ter acesso aos currículos filtrados por bairro.

### 3. Google AdSense
- Espaços reservados em:
  - Entre a 3ª e 4ª vaga da lista.
  - Lateral do mapa de GPS.
  - Sidebar do Admin.

## APIs Necessárias (Planos Grátis)
1. **Supabase**: Banco de dados e autenticação (Auth).
2. **Resend**: Notificações por e-mail (3000/mês grátis).
3. **OpenStreetMap**: GPS e Mapas (Sem custos).
4. **WhatsApp Business API (Opcional)**: Para automação de candidaturas.

## Próximos Passos
1. Conectar ao repositório GitHub.
2. Fazer o deploy na Vercel (Hospedagem Grátis).
3. Configurar as chaves do Supabase no `.env.local`.
